About Me

I am Lizhe Zhang, a second-year master's student in Computer Science at UC San Diego, advised by Prof. Jingbo Shang. I am also fortunate to work with Prof. Thomas Ploetz and Prof. Xia Hu. My primary research focuses on building reliable and self-evolving AI systems, specifically on LLM generalization, uncertainty-aware multi-model collaboration, and diversity-driven synthetic data generation.

Selected Projects

Virtual Reality Gesture Gloves

Arduino-based multi-sensor glove for 3D gesture mapping and obstacle alerts.

  • Designed and built an Arduino-based, multi-sensor glove that captures finger motions using rotary potentiometers.
  • Implemented firmware to map continuous resistance changes to 3D hand gestures in virtual space.
  • Integrated an ultrasonic distance sensor to provide real-world obstacle alerts for user safety.
Video Demo GitHub

Find Your Spot: A Bluetooth Low Energy-based Parking Navigation System

An intelligent parking deck navigation system using ultra-low-power sensors and cloud computation to improve parking-space utilization.

  • Engineered an end-to-end parking navigation system utilizing Bluetooth Low Energy (BLE) beacons.
  • Leveraged RSSI values and path loss models to enable precise location detection and vehicle routing.
  • Optimized parking space allocation with a custom server-side routing algorithm.
